Fitting a Stage-Discharge Rating using a Segmented#

There are several approaches to fitting a stage-discharge rating curve. The first section of this notebook demonstrates the classic approach, which is to use a segmented power law.

Setup model#

Setup a rating model. This make take a minute while the model compiles.

segments = 2
powerrating = PowerLawRating(q=df['q'],
                             h=df['stage'], 
                             q_sigma=df['q_sigma'],
                             segments=segments,
                             prior={'distribution':'uniform'})

then fit the model using variational inference (this will be slower on the first run). Set the number of iterations n such that the model stops after the loss stops decreasing.

with powerrating:
    mean_field = pm.fit(method='advi', n=120_000)
    trace = mean_field.sample(5000)

Simulated Example#

This example uses a simulated rating curve, which allows you to test how different sampling schemes affect the rating curve fit.

First, open the simulated_rating tutorial dataset.

sim_df = data.load('3-segment simulated')
print('The simulated rating contains {} observations'.format(len(sim_df)))

The simulated rating contains 763 observations

This rating contains observations of every 0.01 inch. increment in stage, which is much more than we’d have for a natural rating. Try sampling to n=15 or n=30 and see how that affects the model fit.

Fitting this model is tricky. If the fit appears poor, here are some steps to try.

  1. rerun the model, a new

  2. try increasing the number of iterations.

  3. try a strong prior on the breakpoints. For example, try prior={'distribution':'normal', 'mu':[5, 9.5, 10.5], 'sigma':[1, 1, 0.2]}), which implies we know the true breakpoint within +-0.5 ft.

  4. finally, try fitting the model with another method.

ADVI typically underestimates uncertainty and struggles with multi-modal distributions. NUTS may give better results but will be substantially slower to fit.

Parameterization#

The segmented power law is defined as:

()#\[\begin{align} \log(Q) = a + \sum_{i=1}^{n} b_i \log(x - x_{o,i}) H_i(x - x_{o,i}) \end{align}\]

where $Q$ is a vector discharge,
$n$ is the number of breakpoints in the rating,
$a$ and $b$ are model parameters,
$x$ is a vector of stage observations,
$x_o$ is a vector of breakpoints, and
$H$ is the Heaviside function.

In a standard linear model $b$ represents the slope of the function with respect the input. In the segmented power law $b_o$ is the slope and each subsequent $b_i$ are adjustment to the base slope for each segment.
